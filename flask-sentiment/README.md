# Sentiment analysis

### Prerequisites
You need: `Python3`

Download dataset: `https://www.kaggle.com/kazanova/sentiment140`

Extract dataset to `./src/data/data.csv`

### Installing
`pip install -r requirements.txt`

### Prepare dataset
`python ./src/prepareDataset.py`
### Create model
`python ./src/learn.py`
### Run app
`python ./src/run.py`

Or use docker with `Dockerfile`
